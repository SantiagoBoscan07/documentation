Introduction to CSS

CSS means Cascading Style Sheets and it describes how elements are displayed on screen.

CSS allows you to change the font, the background color, and other elements in a website.

Fonts

In CSS you can modify the style of the font-related elements, including the size, weight, height and the variant.

CSS allows you to use custom types of fonts with font-family, and it is possible to add external fonts to a website.

Colors

You can change the color of fonts and background with CSS. There are different ways to change the color of the elements in a website.

  • The first way is using the name of the color, for example "color:red"
  • The second way is using the RGB (red, green, blue) color model, for example "color:rgb(255, 0, 0)"
  • The third way is using the Hex calculator, for example "color:#ff000"
  • CSS also allows you to change the opacity of the color using the RGBA color model, for example "color:rgba(255, 0, 0, 1)"
  • In the four examples, the color of the font is red
Media query

Media query is used to establish certain styles for different devices. A responsive website utilizes media query to give a better style sheet for tablets and mobile phones.

Media query also changes the style for printed documents and screen readers.

Another important detail about media query is the support of multitple browsers for both computers and mobile devices.

Flexbox

Flexbox is a tool that makes easier to create responsive structures for a website.

CSS uses this tool to give a sense of float and position to the website.

Each flexbox is created within a container, and utilizing other tools from CSS, they can be modified.
